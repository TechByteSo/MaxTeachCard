# MaxTeachCard

Одностраничный сайт-визитка педагога (статический HTML/CSS/JS).

## Локальный запуск

Откройте `index.html` в браузере или поднимите локальный сервер (например, Live Server в VS Code / Cursor), чтобы корректно работали некоторые сценарии.

## Деплой

- Залейте содержимое репозитория на **REG.RU** (или любой статический хостинг с **HTTPS**).
- После привязки домена обновите в `index.html` метатеги Open Graph (`og:url`, `og:image`) на реальные абсолютные URL вашего домена.
- На Linux-серверах учитывайте **регистр** имён файлов и папок (в т.ч. кириллица в `images/` и в ссылках на PDF в `documents/` — имена файлов должны совпадать с реальными на сервере).
- В проекте используйте единые каталоги изображений для контента портфолио:
  - `images/Городские уличные фестивали Москвы 2025/`
  - `images/День города Москва 875/`
  - `images/Участие в БГУ/`
  - `images/Стажировка Волгоград/`

## Портфолио (новый проект в блоке «Показать все»)

1. Скопируйте целый блок `events__card` с соседней карточки внутри `#eventsMoreWrapper` / `.events__more-inner` (перед закрывающим `</div>` этой внутренней обёртки).
2. Задайте у трека уникальный `id` (например `slider12`) и тот же идентификатор в `data-slider` у стрелок и точек; индексы точек — `0 … N-1` по числу слайдов.
3. Обновите пути к изображениям, `alt`, заголовки и подписи в overlay. У разных карточек не должно быть двух треков с одним `id`.

## Контакты на сайте

Обратная связь — статический блок (телефон, почта, ссылка на Telegram). Замените плейсхолдер почты в `index.html` на рабочий адрес перед публикацией.

## Документы (PDF)

Файлы из вкладки «Документы» ожидаются в каталоге `documents/` с теми же именами, что в `href` (кириллица и пробелы учитываются). При отсутствии файлов ссылки на хостинге дадут 404.

## Фавиконы

Сейчас в репозитории есть `favicon.svg`. При появлении `favicon.ico`, PNG и `apple-touch-icon.png` можно снова добавить соответствующие `<link>` в `<head>`.

## Минификация

```bash
npm install
npm run minify
```

Команда пересобирает:
- `styles/style.min.css`
- `styles/slider.min.css`
- `scripts/script.min.js`
